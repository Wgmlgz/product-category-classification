Инструкция:
1) скачать папку
2) открыть в ней консоль
3) установить зависимости (можно только 1 раз на каждой машине)
pip install -r requirements.txt

2) запустить
python -u grab-products.py 0 20

Числа в конце - это стартовая и конечная ссылка, что бы запускать на нескольких машинах нужно поменять ссылки (1 число включительно, 2 невключительно), например: 
python -u grab-products.py 0 50000
python -u grab-products.py 50000 100000

Предлагаю сначала запускать кусками по 50000 (будет работать около 3 дней)
Если программа вылетела (выключился компьютер и тд, то её можно запустить заного, прогресс сохраняется каждые 10 ссылок)
Можно запустить несколько программ на 1 машине, для этого нужно будет скопировать папку, например

папка parser1 -> python -u grab-products.py 0 20 - в папке parser1
папка parser1 -> python -u grab-products.py 20 40 - в папке parser2

(количество папок нужно смотреть на каждой машине отдельно, можно добавлять до тех пор, пока не пойдут ошибки, или время на каждый продукт будет сильно увеличиваться)

содержимое папки:
found.json - наёденные продукты
grab-products.py - программа
queue.txt - очередь с ссылками (одинакавая для всех машин)
requirements.txt - список зависимостей
used.txt - список использованных ссылок

очередь с сылками одинаковая на всех машинах одинакавая (сейчас там около 700000 ссылок), когда все спарсим, я пришлю обновленную, можно её заменить и продолжать
